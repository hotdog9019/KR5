# Кликер-игра (Express.js)

Простая веб-игра на Express.js, где пользователь нажимает кнопку, чтобы получать очки. Прогресс сохраняется на сервере по уникальному ID игрока.

## Функции

- Сохранение счёта в памяти сервера
- Уникальный `playerId` хранится в localStorage
- REST API для получения и обновления счёта
- Простой фронтенд (HTML + CSS + JS)
